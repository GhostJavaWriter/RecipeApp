# RecipeApp
Should be gorgeous app for storing cooking recipes.
Design: Figma, UIKit
Language: Swift
Data storing: CoreData
Features: 
- Create recipes
- Edit recipes
- Share recipes
- Drag and drop to delete a recipe

AppStore link: https://apps.apple.com/app/recipes-keeper/id6450554239
